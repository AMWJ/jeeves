About This Site.

The Jeeves research mission doge.
Image created using NoodlyTime.com.

Because of all of the interactions between different roles and permissions, conference management systems have become a standard benchmark for language and system approaches to security.

We have created this site to demonstrate the capabilities of the Jeeves language for automatically enforcing privacy policies. That's right: this website is running on Python-Jeeves, an embedded domain-specific language implemented in Python. In these Python-Jeeves programs, we can separately implement policies about who can see what values from the rest of the functionality. The main goal of this approach is to decrease the possible places where programmer error can cause the information to be shown inappropriately! We rely on the Python-Jeeves runtime to show the right versions of papers, authors, scores, and reviewer identities to you.

We have also integrated Jeeves with the Django web framework so we can guarantee that interactions with the database do not violate privacy policies.

You may read more about Jeeves here or browse the Python-Jeeves source code here.
